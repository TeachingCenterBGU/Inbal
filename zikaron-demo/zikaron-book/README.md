# זיכרון קולקטיבי — ספר הדגמה ב-Quarto

ספר Quarto לדוגמה בעברית (RTL מלא) שנבנה כפיילוט להצגה למרצים ממדעי הרוח.
מדגים את כל היכולות הרלוונטיות: פרקים, **ציטוטים חיים** המקושרים לביבליוגרפיה,
**ביבליוגרפיה אוטומטית**, הערות שוליים, ציטוטי בלוק (blockquotes), קופסאות
(callouts), טבלאות, ומספור פרקים — הכול מיושר נכון מימין לשמאל.

## מבנה הפרויקט

```
zikaron-book/
├── _quarto.yml        ← הגדרות הספר (כותרת, פרקים, פורמט, ביבליוגרפיה)
├── references.bib     ← מאגר המקורות (BibTeX) — מקור הציטוטים החיים
├── theme.scss         ← עיצוב מותאם + תיקוני RTL + גופנים עבריים
├── index.qmd          ← הקדמה
├── intro.qmd          ← מבוא
├── 01-misgarot.qmd    ← פרק 1
├── 02-atarim.qmd      ← פרק 2
├── 03-tarbut.qmd      ← פרק 3
└── references.qmd     ← עמוד המקורות (הביבליוגרפיה נוצרת כאן אוטומטית)
```

## איך מעבדים (render)

צריך את Quarto מותקן (גרסה 1.4 ומעלה): <https://quarto.org/docs/get-started/>

```bash
# תצוגה חיה עם רענון אוטומטי בזמן עריכה:
quarto preview

# עיבוד מלא לאתר HTML (נוצר בתיקייה _book/):
quarto render
```

פותחים את `_book/index.html` בדפדפן — וזהו ספר אינטרנטי מלא עם ניווט,
חיפוש וסרגל צד.

## הדברים שכדאי להראות למרצה

| יכולת | איפה רואים אותה |
|---|---|
| ציטוט חי בגוף הטקסט | `[@nora1989]` ← לחיצה מקפיצה לערך המלא |
| ביבליוגרפיה אוטומטית | עמוד "מקורות" — נבנה מ-`references.bib` |
| הערת שוליים | בהקדמה (`[^pref-1]`) |
| ציטוט בלוק | הציטוט מנורה בפרק 2 |
| קופסאות מודגשות | callout-note / tip / important בכל הפרקים |
| טבלה | טבלת ההשוואה בפרק 3 |
| RTL מלא | כותרות, רשימות, טבלאות, מספור — הכול מימין לשמאל |

## הערה על הציטוטים

הציטוטים משתמשים בסגנון Chicago author-date (ברירת המחדל). אפשר להחליף
לכל סגנון CSL אחר (כולל סגנונות עבריים) ע"י הוספת שורה ב-`_quarto.yml`:

```yaml
csl: my-style.csl
```

מאגר סגנונות: <https://www.zotero.org/styles>

## הערה על PDF

ב-`_quarto.yml` יש בלוק `pdf` מושבת (בהערה). להפעלתו צריך מנוע XeLaTeX
עם גופן עברי מותקן — מומלץ להתחיל מ-HTML ולהוסיף PDF בהמשך.
